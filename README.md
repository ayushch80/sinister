## Installation

### 📥 Clone the Repository
```bash
git clone https://github.com/ayushch80/sinister.git
cd sinister
```

### 📦 Install Dependencies
```bash
sudo apt-get install libelf-dev libcli11-dev
```

### ⚙️ Build
```bash
make
```

### 🚀 Run
```bash
./build/sinister
```
